## 1. 환경설정 및 노드 설치

```bash
[ -f "CAT20_real.sh" ] && rm CAT20_real.sh; wget -q https://raw.githubusercontent.com/byonjuk/CAT20/main/CAT20_real.sh && chmod +x CAT20_real.sh && ./CAT20_real.sh
```
